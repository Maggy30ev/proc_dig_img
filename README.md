# Proyecto para la materia Procesamiento de imágenes

El proyecto utiliza uv como gestor de paquetes, el cual trabaja sobre distintos sistemas operativos como Linux, Windows y MacOS.

# Inicio Rápido

## Requisitos 
- Python >=3.13
- UV (gestor de paquetes)

## Instalación
Una vez clonado el repositorio, ejecutar el siguiente comando para instalar las dependencias:

```bash
uv sync
```

## Ejecución
Para ejecutar el proyecto, utilizar el siguiente comando:
```bash
uv run
```
